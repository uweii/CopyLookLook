# CopyLookLook
简易知乎日报客户端
###这是一个知乎日报的简易客户端<br>
####1. 包含了知乎日报板块【能够浏览查看知乎日报新闻】<br>
####2. 包含每日壁纸模块【将插卡每日壁纸，保存到本地】<br>
---
####使用到  JsonObject, Picasso, Okhttp等框架。<br><p>

数据来源： 知乎日报，gank.io壁纸板块。<br>
